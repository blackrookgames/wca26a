HOW TO USE

Every terminal command follows this basic format

interpreter wca26a options args

Example

python3 wca26a ass --output pong_ripoff.a26 pong_ripoff.asm

ass

Assembles a ROM

ass [--help] [--output value] [--intmd value] [--flags value] [--nocase] source

sourceInput path; (*.asm)
--helpDisplays help
--output valueOutput path; (*.a26;*.bin)
-o value
--intmd valueIntermediate path; this can be used to see what source looks like after macros and labels are resolved.
--flags valueComma separated list of flags; these can be used during preprocessing to determine what code to include or exclude
--nocaseIf specified, case-sensitivity will not be enforced

NOTE: Bank switching not currently supported

Example

python3 wca26a ass --flags DEBUG,TESTING --output pong_ripoff.a26 pong_ripoff.asm

dis

Disassembles a ROM

ass [--help] [--output value] [--nocmn] rom

romROM path (*.a26;*.bin)
--helpDisplays help
--output valueOutput path; (*.asm)
-o value
--nocmnIf specified, common definitions (VSYNC, COLUP0, etc) will not be used

NOTE: Bank switching not currently supported

Example

python3 wca26a dis --nocmn -o space_game_disassembly.asm space_game.bin
